1) Difference between map(), forEach(), filter() & find()

1) map() goes through all element and returns something done.
2) forEach() approaches each element (if you no need returns).
3) filter() on the other hand goes through all the elements and returns only those that match the criteria.
4) find() condition will only go to the first element and not the rest.

2) About Local Storage, Section Storage & Cookies

a) The data is set and saved in local storage and remains the same even when a new tab is opened.
b) On the other hand, the data set and get are also removed in the section storage, but the page is refreshed as soon as the data is removed.
c) Through cookies, data is saved in the storage of the browser, which is deleted after a certain period of time.

3) Event Loop in JavaScript

Java Script is a single threaded language. It always runs in synchronous way. That is, after calling a function, another function will execute. If one function calls another function, it will call all of them and then go to the next function. In between these functions if asynchronous calls are made like settimeout, fetch etc. then they are sent to the callback queue from where the event loop goes to the callstack and is called. This is basically the event loop.
